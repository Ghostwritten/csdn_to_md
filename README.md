# csdn 文章批量转换markdown格式下载至本地

---
## 下载

```bash
$ git clone https://github.com/Ghostwritten/csdn_to_md.git 
```

## 配置

chrome浏览器登陆csdn平台，按"F12"找到自己网页cookie,选择部分cookie内容复制至csdn_to_md.py脚本109行。
